Introduction

Futuwear - The next generation wearable smart clothing from Aalto University's Protopaja. It utilizes enhanced 9 DoF IMU technology, that maps the user's body movements with pristine accuracy. Scroll down to learn more of this revolutionary technology.

How it works

  • IoT Microcontroller

    The Futuwear V2 uses the ESP32 microcontroller as the main processor. It has a built-in WiFi-module that enables it to communicate directly to a host server and control a 3D-model.

  • IMU sensors

    The Futuwear V2 supports MPU-9250 9 DoF (Degrees of Freedom) IMUs (Inertial Measurement Unit), which contain an accelerometer, a gyroscope and a magnetometer to ensure as precise positional sensing as possible with today's technology.

  • Web interface

    The device connects to a web server, which displays the orientation and movements of the user in realtime. The server connects to the device and to the user through the websocket protocol. This makes it possible to expand the usecases of the product. For example to remote control a robotic arm.

Background

The Futuwear V2 is based on the last year's Futuwear V1 smart clothing. The insights of last year's team have been taken into account. Their prototype used flex sensors to measure the user's movements, and later send that data over Bluetooth.
