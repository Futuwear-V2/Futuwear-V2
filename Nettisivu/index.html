Introduction

Futuwear V2 - The next generation wearable smart clothing from Aalto University's Protopaja. It utilizes enhanced 9 DoF IMU technology, that maps the user's every movement with pristine accuracy. Scroll down to learn more of this revolutionary technology.

How it works

  • IoT Microcontroller

    The Futuwear V2 uses the ESP32 microcontroller as the main processor. It has a built-in WiFi-module that enables it to communicate directly to a host server and control a 3D-model.

  • IMU sensors

    The Futuwear V2 supports MPU-9250 9 DoF (Degrees of Freedom) IMUs (Inertial Measurement Unit), which contain an accelerometer, a gyroscope and a magnetometer to ensure the most precise positional sensing possible with today's technology.

  • Web interface

    The device connects to a web server, which displays the orientation and movements of the user in real-time. The connection is established using the websocket protocol. By sending the data straight off the MCU this way, it opens up countless possibilities and usecases for the technology.

Background

The Futuwear V2 is based on the 2016 Futuwear V1 smart clothing. The insights of last year's team have been taken into account. Their prototype used flex sensors to measure the user's movements, and later send that data over Bluetooth to a host computer.

They aimed to create a sleek and comfortable smart clothing, that would serve the user well for a long time. We wanted to carry that same view with us throughout the entire project and create something equally unique.
